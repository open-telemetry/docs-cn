这是 `开放遥测` 的 Go 文档。 `开放遥测` 是一个可观测性框架 -- 包含API、SDK 和一些生成和收集应用程序遥测数据如指标、日志和链路信息的工具。 本文档旨在帮助您了解在 Go 语言中如何开始使用 `开放遥测`。

## 状态

OpenTelemetry Go 主要功能组件的现状如下：

| Tracing | Metrics | Logging |
| ------- | ------- | ------- |
| 稳定的  | 内部测试(Alpha)   | 还未实现 |
